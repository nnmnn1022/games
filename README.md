# Game Showcase Website

A modern, interactive, and responsive "Game Showcase" landing page built with HTML, CSS, and JavaScript.

## Features
- **Modern Design**: High-impact hero section with vibrant gradients and glassmorphism.
- **Responsive Layout**: Fully responsive grid using CSS Flexbox and Grid.
- **Dynamic Content**: Game cards are dynamically generated from JavaScript data.
- **Vercel-Ready**: Simple structure for instant deployment on Vercel.

## Deployment on Vercel
1. Install the Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory to deploy instantly.
3. Or, push this code to a GitHub repository and connect it to your Vercel account for automatic deployments.

## Project Structure
- `index.html`: Semantic HTML5 structure.
- `style.css`: Modern styling with CSS variables and animations.
- `script.js`: Interactive elements and dynamic content generation.
- `assets/`: Directory for images, icons, and other assets.
